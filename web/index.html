OnchainOS 集成 (5 模块)

Wallet

核心身份

Agentic Wallet 作为 Agent 身份和资金管理。TEE 安全环境保护私钥。

DEX 聚合器

交易引擎

500+ 流动性来源路由。HMAC-SHA256 签名认证。V6 API 实时报价。

Market 市场

数据层

实时价格数据，驱动告警服务和定价决策。代币详情和价格查询。

Security 安全

信任层

代币扫描（蜜罐、铸币权、代理合约）+ 合约审批风险检测。

x402 支付

支付协议

ERC-20 转账构造 + 链上支付验证。支付门控服务交付。

Uniswap 集成 (2 技能)

Trading

路由对比

替代 swap 路由。与 OnchainOS DEX 对比，选择净输出最高的执行路径。不同滑点策略的竞争报价。

Pay-Any-Token

灵活支付

接受任意代币支付服务费，通过 Uniswap 自动兑换为 USDT 结算。链上 allowance 检查 + approve 构造。
